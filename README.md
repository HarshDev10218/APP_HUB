🎮 App Hub

A Python-based multi-application hub built with Tkinter and Pygame. This project combines a scientific calculator with several mini-games inside one desktop application. Users can switch between applications from a central home screen.

✨ Features
🧮 Scientific Calculator

Includes support for:

Basic arithmetic operations (+, -, *, /)
Exponents (^)
Square roots
Logarithms
Trigonometric functions:
sin, cos, tan
asin, acos, atan
Factorials
Mathematical constants:
π (Pi)
e (Euler's number)
🐍 Snake Game

Classic Snake arcade game with:

Smooth movement controls
Food spawning system
Collision detection
Score tracking
High score saving (highscore.txt)
Start and game-over menus

Controls

Key	Action
↑ ↓ ← →	Move Snake
S	Start Game
H	View High Score
R	Restart
Q	Quit
🚀 Space Invaders

A simple Space Shooter featuring:

Multiple enemy waves
Increasing difficulty
Laser shooting mechanics
Score counter
Restart and exit options

Controls

Key	Action
← →	Move
Space	Fire Laser
R	Restart
Q	Quit
🎲 Pig Dice Game

Two-player dice game where players race to reach 100 points.

Rules

Press R to roll.
Rolling a 1 loses the turn score.
Press H to hold and bank points.
First player to reach 100 points wins.

⚙️ Installation
1. Clone the repository
git clone https://github.com/yourusername/app-hub.git
cd app-hub
2. Install dependencies
pip install pygame

(Tkinter is included with most Python installations.)

3. Run the program
python main.py
